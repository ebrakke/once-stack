Database schema decisions

Updated Apr 25, 2026

Edit

After the review session, we settled on the following schema for the notes table:

CREATE TABLE notes (
  id        TEXT PRIMARY KEY,
  title     TEXT NOT NULL,
  body      TEXT,
  slug      TEXT NOT NULL UNIQUE,
  created   INTEGER NOT NULL,
  updated   INTEGER NOT NULL
);

Key decisions:

  • Use TEXT IDs (slug-based) instead of auto-increment — makes URLs cleaner and simplifies import/export
  • Store created/updated as Unix timestamps (INTEGER) to avoid timezone headaches
  • No foreign keys for now — this is a single-entity app

How to start a compost pile

Updated Apr 20, 2026

Edit

Composting is nature's way of recycling. Here's a simple guide to get started.

What you need

  • A bin or an open pile location
  • Browns (dried leaves, cardboard, wood chips)
  • Greens (kitchen scraps, grass clippings, coffee grounds)
  • Water + air (turn every few days)

Layer ratio

Aim for roughly 3 parts browns to 1 part greens. Too many greens and it gets smelly. Too many browns and it takes forever.

What NOT to compost

  • Meat, dairy, or oily foods (attracts pests)
  • Diseased plants
  • Weeds that have gone to seed

In ~3–6 months you'll have dark, crumbly compost ready for the garden.

Sourdough starter recipe

Updated Apr 14, 2026

Edit

A sourdough starter is a living culture of wild yeast and bacteria. Here's how to make one from scratch.

Day 1

Mix 50g whole wheat flour + 50g water in a jar. Cover loosely. Leave at room temperature.

Days 2–7

Each day, discard half, then add 50g flour + 50g water. Stir. By day 5–7 it should double in size within 6–8 hours of feeding — that's when it's ready to bake.

Maintenance

  • Store in fridge if feeding less than once a week
  • Feed 1:1:1 ratio (starter : flour : water) before baking
  • If hooch (dark liquid) forms on top, pour it off — your starter is hungry
